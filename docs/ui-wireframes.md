# RouteFusion UI Wireframes

These wireframes now follow the Uber-inspired persistent map layout requested for the RouteFusion redesign.

## Shared Navigation

```text
+----------------------------------------------------------------------------------------------------------------+
| RouteFusion | Home | Parcel Request | Ride Request | Captain Corner | Live Map | Dashboard | Profile | Demo On |
+----------------------------------------------------------------------------------------------------------------+
```

## Global Workspace

```text
+-------------------------------+------------------------------------------------------------------------+
| Left Panel (30%)              | Persistent Map (70%)                                                   |
| Route-specific content        | Google Maps / Fallback Map                                             |
| Search, forms, captain info   | Markers, routes, live captain marker, route dock                       |
| Dashboard cards, profile      |                                                                        |
+-------------------------------+------------------------------------------------------------------------+
```
